gensym :: String -> String Source #

Generate a fresh variable based on the given one. The resulting variable has the form "xn" is a "uniqueness suffix".

Warning: This function relies on unsafePerformIO to avoid putting otherwise pure code into the IO monad.

Examples:

>>> gensym "X"
"X#2"

>>> gensym "X"
"X#3"

>>> gensym "Z"
"Z#4"
