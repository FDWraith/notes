Types

Description

Types used in protoScheme.

This module defines the type language of protoScheme. Types have the following syntax:

<BaseType> ::= Integer
             | Real
             | Boolean

<Type> ::= <BaseType>
         | (-> <Type>+)
         | (Pair-of <Type> <Type>)
         | (List-of <Type>)
         | <TypeVariable>
